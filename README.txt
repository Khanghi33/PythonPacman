How to run this code:
1. Make sure you has installed python and pygame library in your computer
2. Open file main.py and run python file
3. Click button PLAY and there are 6 option. But just option level you can play, the others are just for test
4. When use choose LEVEL6 use need to enter your username. This game will record your score by your username. Go to leaderboard to see your rank
Extend: If your want to minimize or maximize your screen go to file config.py and modify the Width (please do not modify the height)
